42 /*!
43 
44 \brief LOG and sLOG for development and debugging
45 
46 This is a short description of how to use \ref LOG and \ref sLOG for rapid
47 development of modules with debug output, and how to **keep it afterwards**.
48 
49 There are two classes Logger and SpacingLogger, but one does not use these
50 directly.
51 
52 Instead there are the macros: \ref LOG and \ref sLOG that can be used as such:
53 \code
54 LOG << "This will be printed with a newline";
55 sLOG << "Print variables a" << a << "b" << b << "c" << c;
56 \endcode
57 
58 There macros only print the lines if the boolean variable **debug** is
59 true. This variable is searched for in the scope of the LOG, which means it can
60 be set or overridden in the function scope, the class scope, from **inherited
61 classes**, or even the global scope.
62 
63 \code
64 class MyClass
65 {
66  static constexpr bool debug = true;
67 
68  void func1()
69  {
70  LOG << "Hello World";
71 
72  LOG0 << "This is temporarily disabled.";
73  }
74 
75  void func2()
76  {
77  static constexpr bool debug = false;
78  LOG << "This is not printed any more.";
79 
80  LOG1 << "But this is forced.";
81  }
82 };
83 \endcode
84 
85 There are two variation of \ref LOG and \ref sLOG : append 0 or 1 for
86 temporarily disabled or enabled debug lines. These macros are then \ref LOG0,
87 \ref LOG1, \ref sLOG0, and \ref sLOG1. The suffix overrides the debug variable's
88 setting.
89 
90 After a module works as intended, one can just set `debug = false`, and all
91 debug output will disappear.
92 
93 ## Critique of LOG and sLOG
94 
95 The macros are only for rapid module-based development. It cannot be used as an
96 extended logging system for our network framework, where logs of network
97 execution and communication are collected for later analysis. Something else is
98 needed here.
99 
100  */
101 class Logger
102 {
103 private:
104  //! collector stream
105  mem::safe_ostringstream oss_;
106 
107 public:
108  //! mutex synchronized output to std::cout
109  static void Output(const char* str);
110  //! mutex synchronized output to std::cout
111  static void Output(const std::string& str);
112  //! mutex synchronized output to std::cout
113  static void Output(const mem::safe_string& str);
114 
115  Logger();
116 
117  //! output any type, including io manipulators
118  template <typename AnyType>
119  Logger& operator << (const AnyType& at) {
120  oss_ << at;
121  return *this;
122  }
123 
124  //! destructor: output a newline
125  ~Logger();
126 };

234 //! Logging helper to print arrays as [a1,a2,a3,...]
235 template <typename T, size_t N>
236 log_stream& operator << (log_stream& os, const std::array<T, N>& data) {
237  os << '[';
238  for (typename std::array<T, N>::const_iterator it = data.begin();
239  it != data.end(); ++it)
240  {
241  if (it != data.begin()) os << ',';
242  os << *it;
243  }
244  os << ']';
245  return os;
