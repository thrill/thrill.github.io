OurConcurrentQueue< T, Allocator > Class Template Reference

Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class thrill::common::OurConcurrentQueue< T, Allocator >

This is a synchronized queue, similar to std::queue and moodycamel's concurrent_queue, except that it uses mutexes for synchronization.

This implementation is for speed and interface testing against a lock-free queue.
