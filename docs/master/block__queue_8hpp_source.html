35 /*!
36  * A BlockQueue is a thread-safe queue used to hand-over Block objects between
37  * threads. It is currently used by the Multiplexer to queue received Blocks and
38  * deliver them (later) to their destination.
39  *
40  * The BlockQueue itself is also a BlockSink (so one can attach a BlockWriter to
41  * it). To read items from the queue, one needs to use a BlockReader
42  * instantiated with a BlockQueueSource. Both are easily available via
43  * GetWriter() and GetReader(). Each block is available only *once* via the
44  * BlockQueueSource.
45  */
