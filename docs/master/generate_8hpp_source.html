70 /*!
71  * Generate is a Source-DOp, which creates a DIA of given size using a
72  * generator function. The generator function called for each index in the range
73  * of `[0,size)` and must output exactly one item.
74  *
75  * \param ctx Reference to the Context object
76  *
77  * \param size Size of the output DIA
78  *
79  * \param generate_function Generator function, which maps `size_t` from

81  *
82  * \ingroup dia_sources
83  */

86  const GenerateFunction& generate_function) {

93
