147 /*!
148  * Construct reader for given path uri. Range is the byte range [b,e) inside the
149  * file to read. If e = 0, the complete file is read.
150  *
151  * For the POSIX SysFile implementation the range is used only to seek to the
152  * byte offset b. It allows additional bytes after e to be read.
153  *
154  * For the S3File implementations, however, the range[b,e) is used to determine
155  * which data to fetch from S3. Hence, once e is reached, read() will return
156  * EOF.
