23 /*!
24  * A ring (circular) buffer of static (non-growing) size allocated. This data
25  * structure is mostly used by the DIA.Window() transformation.
26  *
27  * Due to many modulo operations with capacity_, the capacity is rounded up to
28  * the next power of two, even for powers of two! This is because otherwise
29  * size() == end - begin == 0 after filling the ring buffer, and adding another
30  * size_ member requires more book-keeping.
31  */
32 template <typename Type, typename Allocator = std::allocator<Type> >
33 class RingBuffer
