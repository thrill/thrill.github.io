37 /*!
38 
39 \brief LOG and sLOG for development and debugging
40 
41 This is a short description of how to use \ref LOG and \ref sLOG for rapid
42 development of modules with debug output, and how to **keep it afterwards**.
43 
44 There are two classes Logger and SpacingLogger, but one does not use these
45 directly.
46 
47 Instead there are the macros: \ref LOG and \ref sLOG that can be used as such:
48 \code
49 LOG << "This will be printed with a newline";
50 sLOG << "Print variables a" << a << "b" << b << "c" << c;
51 \endcode
52 
53 There macros only print the lines if the boolean variable **debug** is
54 true. This variable is searched for in the scope of the LOG, which means it can
55 be set or overridden in the function scope, the class scope, from **inherited
56 classes**, or even the global scope.
57 
58 \code
59 class MyClass
60 {
61  static constexpr bool debug = true;
62 
63  void func1()
64  {
65  LOG << "Hello World";
66 
67  LOG0 << "This is temporarily disabled.";
68  }
69 
70  void func2()
71  {
72  static constexpr bool debug = false;
73  LOG << "This is not printed any more.";
74 
75  LOG1 << "But this is forced.";
76  }
77 };
78 \endcode
79 
80 There are two variation of \ref LOG and \ref sLOG : append 0 or 1 for
81 temporarily disabled or enabled debug lines. These macros are then \ref LOG0,
82 \ref LOG1, \ref sLOG0, and \ref sLOG1. The suffix overrides the debug variable's
83 setting.
84 
85 After a module works as intended, one can just set `debug = false`, and all
86 debug output will disappear.
87 
88 ## Critique of LOG and sLOG
89 
90 The macros are only for rapid module-based development. It cannot be used as an
91 extended logging system for our network framework, where logs of network
92 execution and communication are collected for later analysis. Something else is
93 needed here.
94 
95  */
96 class Logger
97 {
98 private:
99  //! collector stream
100  mem::safe_ostringstream oss_;
101 
102 public:
103  //! mutex synchronized output to std::cout
104  static void Output(const char* str);
105  //! mutex synchronized output to std::cout
106  static void Output(const std::string& str);
107  //! mutex synchronized output to std::cout
108  static void Output(const mem::safe_string& str);
109 
110  Logger();
111 
112  //! output any type, including io manipulators
113  template <typename AnyType>
114  Logger& operator << (const AnyType& at) {
115  oss_ << at;
116  return *this;
117  }
118 
119  //! destructor: output a newline
120  ~Logger();
121 };

125  * operator<<. Depending on the real parameter the output may be suppressed.
126  */
127 class SpacingLogger
128 {
129 private:
130  //! true until the first element it outputted.
131  bool first_ = true;
132 
133  //! collector stream
134  mem::safe_ostringstream oss_;
135 
136 public:
137  SpacingLogger();
138 
139  //! output any type, including io manipulators
140  template <typename AnyType>
141  SpacingLogger& operator << (const AnyType& at) {
142  if (!first_) oss_ << ' ';
143  else first_ = false;

145  oss_ << at;

148  }
